# RELIANCE Stock Price Prediction using Machine Learning

## Project Overview
This project predicts stock price movement using ML models on RELIANCE dataset.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Moving Average (MA10)
- Daily Return
- Volume-based indicators

## Outputs
- Stock Trend Graph
- Prediction vs Actual Graph
- Feature Importance
- Accuracy & R2 Score

## Tools
Python, Pandas, Scikit-learn, Matplotlib
